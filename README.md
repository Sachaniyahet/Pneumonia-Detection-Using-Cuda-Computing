# Pneumonia-Detection-Using-Cuda-Computing
RapidRespiro AI is an advanced medical imaging platform that automates pneumonia detection strictly from chest X-rays. It uses an ensemble of DenseNet121 and ResNet34 architectures, optimized via CUDA. Featuring Grad-CAM heatmaps for explainable AI, it delivers precise insights through a sleek, high-contrast "cyborg" interface.
